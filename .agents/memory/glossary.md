# AAC V3 Glossary

- **AAC**: Antigravity Agent Core.
- **AGY / agy**: The outer-wrapper Command Line Interface tool for managing workspace automation.
- **Prompt Caching**: Restructuring file layouts so large prompt contexts (like code summaries) remain static across multiple invocations, reducing latency and cost.
- **Locking**: The process of reserving a specific module or component file path for single-agent modification to prevent concurrent merge conflicts.
- **ADR**: Architectural Decision Record.
